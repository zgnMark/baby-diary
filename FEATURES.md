# 📖 功能说明文档

## 1. 概述

宝宝成长记录 (BabyDiary) 是一款专为新手父母设计的宝宝日常记录应用，参考美柚、小红书等母婴平台的功能设计，帮助父母记录宝宝成长的每一个珍贵瞬间。

## 2. 功能模块

### 2.1 首页

**功能**：
- 宝宝信息卡片展示（头像、姓名、年龄）
- 今日概览（喂养次数、睡眠时长、排便次数）
- 最近记录时间线
- 快速添加记录按钮

**界面元素**：
- 宝宝头像（大圆卡片）
- 宝宝姓名（使用可爱字体）
- 计算年龄（自动根据出生日期计算）
- 今日统计卡片（喂养、睡眠、排便）
- 最近10条记录列表

### 2.2 记录页

支持6种记录类型：

#### 2.2.1 喂养记录
- **喂养方式**：母乳、奶粉、辅食
- **母乳记录**：
  - 喂奶侧（左侧/右侧/双边）
  - 时长（分钟）
- **奶粉记录**：
  - 奶量（ml）
- **辅食记录**：
  - 食物描述
- **时间**：自动记录当前时间

#### 2.2.2 睡眠记录
- 开始时间
- 结束时间
- 睡眠时长（自动计算）
- 睡眠状态（深睡/浅睡/醒来）

#### 2.2.3 排便记录
- 类型：大便/小便
- 状态：正常/稀/干
- 备注说明
- 时间

#### 2.2.4 生长记录
- 日期
- 身高（cm）
- 体重（kg）
- 头围（可选 cm）

#### 2.2.5 疫苗记录
- 疫苗名称
- 接种日期
- 接种部位
- 备注

#### 2.2.6 里程碑
- 标题（如：第一次抬头、第一次翻身）
- 日期
- 描述

### 2.3 统计页

**功能**：
- 身高体重生长曲线图
- 统计摘要

**图表说明**：
- 使用 Recharts 绘制折线图
- 粉色线条表示身高
- 绿色线条表示体重
- X轴：日期
- Y轴：数值

### 2.4 设置页

**功能**：
- 宝宝信息管理
  - 姓名
  - 出生日期
  - 性别
  - 头像（本地选择/拍照）
- LLM 配置
  - 模型选择（OpenAI/Claude/通义千问/自定义）
  - API Key
  - 自定义 Endpoint
- 数据管理
  - 导出 JSON
  - 导入 JSON
- 关于信息

### 2.5 AI 助手（Page Agent）

**位置**：右上角消息图标

**功能**：
- 自然语言记录
- 智能问答
- 数据查询

**支持的命令**：
```
"宝宝吃了100ml奶粉"     → 自动添加奶粉记录
"母乳20分钟"           → 自动添加母乳记录
"宝宝睡觉了"           → 开始记录睡眠
"今天吃了几次？"       → 查询今日喂养次数
"宝宝多大了？"         → 查询宝宝年龄
```

**工作原理**：
1. 用户输入自然语言
2. 本地规则匹配识别意图
3. 如果配置了 LLM，调用 LLM 智能处理
4. 自动调用 Supabase API 添加记录

## 3. 数据存储

### 3.1 本地存储 (localStorage)
- LLM 配置信息
- 页面状态

### 3.2 云端存储 (Supabase)
- 宝宝信息
- 所有记录数据

**数据安全**：
- Row Level Security (RLS) 策略
- 家庭成员共享访问

- 表单弹窗填写

- 长按记录卡编辑/删除

### 4.3 响应式设计
- 移动端优先
- 适配手机屏幕
- 触摸友好

- React 18 函数组件
- Hooks 状态管理

├── HomePage       (首页)

## 7. 常见问题

### Q: 如何添加第一条记录？
A: 进入"记录"页面，点击对应的记录类型标签，填写表单后保存。

### Q: 数据会自动保存吗？
A: 是的，每次添加/编辑/删除记录后，数据会自动同步到云端。

### Q: AI 助手怎么使用？
A: 点击右上角消息图标，可以用自然语言描述要记录的内容，如"宝宝吃了100ml奶粉"。

### Q: 如何导出数据？
A: 进入"设置"页面，点击"导出数据"按钮，会下载 JSON 文件。

### Q: 可以多人同时使用吗？
A: 可以，只要使用同一个 Supabase 项目，家庭成员都可以查看和添加记录。
